Help with the cases page

Cases are used to record findings of investigations and to notify the OPSS about unsafe and non-compliant products.

You can search for a case using the keywords that describe it and by filtering the results. You can also filter all case records without a keyword search.

You can create a new case or add a product to an existing case.
