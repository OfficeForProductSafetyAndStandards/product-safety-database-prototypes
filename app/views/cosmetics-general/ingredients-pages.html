Ingredients pages

Here are the initial designs - not finalised - for the new ingredients pages (to be inserted into the current user journey).

Main design(s)

  • Add the ingredients form
    We insert this page into the standard flow. There is contextual text refering to the ingredients of this product or this item. Click, Save and continue.
    • Do you want to add another? We ask if the user wishes to add another ingredient for the product of the item. (Clicking No, allows the user to exit the loop). Click, Yes.
    • Add another ingredient form This version shows the start of the list of ingredients already added to the product or item. Click, Yes (to see a longer list).
    • Add another ingredient form (with longer list) This version shows a list of ingredients already added to the product or item.

Main design(s) - when editing

Showing the 'remove' link which shows on this page only when editing drafts, re-taking tasks, or clicking through tasks that are 'In progress'.

And showing the 'add a new ingredient' page, which is inserted into the flow, at the end of paging through the ingredients pages, only when editing drafts, re-taking tasks, or clicking through tasks that are 'In progress'.

  • Add the ingredients form - with link
    When clicking Save and continue, the user simply pages onto the next ingredient in the list. We should update the title tags to read, 'Add the ingredients - ingredient [number]'.
    • Do you want to remove it? Clicking No, will return the user back to the previous page. electing no, takes the user onto the next page in the current journey. Click, Yes
    • Removed - confirmed This standalone page is required as the ingredient may be the last in the list of ingredients (making inline success banners inappropriate). No back link. Contextual text "... from this product or this item". Click Continue; the user now goes to the next ingredient form page shown in the ingredients list. If this was the last ingredient the user goes to the 'Add an ingredient' page (below)
  • Do you want to add a new ingredient to this product/item
    No, goes to the next page in the current journey, Yes, goes to a blank ingredient page to append another ingredient to the list (without a remove link). Contextual text "... to this product or to this item". The new ingredient is added immediately (before the end of the task).
