Search cosmetic product notifications for notifications which include a specific ingredient.
Only products notified after 2 October 2022 are searchable for ingredients.

Help with the ingredients search

For best results, search for an individual ingredient. This will search cosmetic product notifications for products which use it.

The "match type" filter option allows you to search for exact name matches or for ingredient names which feature one or more words of the name you are searching for and returns more general search results.

You can use the filter options to focus the search results. You can also choose to display the results grouped together by notifications which share the same Responsible Person.
